# Enzora – Smart Wound Patch Website

Official company website for **Enzora**, a smart medical startup developing 
infection-detecting bandages using biosensing technology.

## 🔗 Live Site
> Coming soon – deployed on Vercel

## 🧱 Tech Stack
- [Next.js](https://nextjs.org/) – React framework
- [Tailwind CSS](https://tailwindcss.com/) – Styling
- [Firebase Firestore](https://firebase.google.com/) – Order storage
- [Vercel](https://vercel.com/) – Deployment
